# DB & API Bug Analysis

Анализ возможных причин бага на основе схемы клиент-сервисной архитектуры.

## Scope
- Анализ структуры БД
- Проверка связей между сущностями
- Поиск причин некорректного поведения

## Что сделано
- Выделены возможные причины бага:
  - на уровне базы данных
  - SQL-запросов
  - backend логики
  - API

## Подход
Причины сгруппированы по уровням системы для системного анализа проблемы.
